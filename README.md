# AI Workout (AiWorkaut) 🏋️‍♂️

**AI Workout** — это современное приложение для отслеживания тренировок на iOS, разработанное для тех, кто использует искусственный интеллект для составления программ тренировок. Приложение ориентировано на удобство ввода данных и визуализацию прогресса.

## ✨ Основные возможности

- **Smart Plan Parser:** Просто скопируйте тренировочный план, сгенерированный нейросетью (ChatGPT, Claude и др.), и вставьте его в приложение. Оно само распознает упражнения, подходы, повторения и веса.
- **Dynamic Island & Live Activities:** Отслеживайте текущий подход прямо на "островке" или экране блокировки. Отмечайте выполненные сеты в один тап, не заходя в приложение.
- **Удобный ввод данных:** Забудьте о клавиатуре. Выбирайте веса и повторения с помощью нативных "крутилок" (Pickers).
- **Экспорт в Markdown:** Сохраняйте историю своих тренировок в красивом табличном формате Markdown для Obsidian, Notion или других заметочников.
- **Локализация:** Полная поддержка русского языка для календаря и интерфейса.

## 🛠 Технологии

- **SwiftUI:** Современный декларативный интерфейс.
- **SwiftData:** Надежное хранение данных на устройстве.
- **ActivityKit:** Интеграция с Dynamic Island и Live Activities.
- **Regular Expressions:** Мощный парсинг текстовых планов тренировок.

## 🚀 Как начать

1. Сгенерируйте тренировочный план в любом AI-чате (рекомендуемый формат — Markdown таблица).
2. Нажмите иконку 🪄 (Smart Parser) на главном экране.
3. Вставьте текст и нажмите "Импортировать".
4. Нажмите кнопку **Play** внизу экрана, чтобы запустить режим тренировки на Dynamic Island.

## 📝 Пример поддерживаемого формата для парсинга:

```markdown
| Упражнение | Подходы x Повторения | Вес | Примечания |
|---|---|---|---|
| Жим лежа | 3 x 10 | 60 кг | Контролируй опускание |
| Приседания | 3 x 12 | 80 кг | Спина прямая |
```
